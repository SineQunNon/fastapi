## 기능 요구 사항
### 로그인 
- [x] 아이디와 비밀번호로 로그인한다.
  - 아이디는 이메일 형식이다.
  - [x] 예외 처리
    - 이메일 형식이 짧은 경우
    - 입력하지 않은 경우
    - 유효하지 않은 경우 예외 발생
      ```
      *올바른 이메일 주소 형식을 입력해주세요(예: example@example.com)
      ```
  - 비밀번호는 8자 이상, 20자 이하, 대문자, 소문자, 숫자, 특수문자를 각각 최소 1개 포함해야 한다.
  - [x] 예외 처리
    - 비밀번호 확인 유효성을 통과 못했을 경우
      ```
      *비밀번호는 8자 이상, 20자 이하이며, 대문자, 소문자, 숫자, 특수문자를 각각 최소 1개 포함해야 한다.
      ```
    - 비밀번호 입력하지 않은 경우
      ```
      *비밀번호를 입력해주세요
      ```
  - [x] 로그인 실패 시
    ```
    * 아이디 또는 비밀번호를 확인해주세요.  
    ```
### 회원가입
- [ ] 프로필 사진(선택), 이메일, 비밀번호, 비밀번호 확인, 닉네임으로 회원가입한다.
  - [ ] 이메일 유효성 검증
    - 이메일이 비어있는 경우
    - 이메일 형식이 너무 짧은 경우, 입력하지 않은 경우, 유효하지 않은 경우
    - 중복된 이메일인 경우
  - [ ] 비밀번호 유효성 검증
    - 8자 이상, 20자 이하, 대문자, 소문자, 숫자, 특수문자를 각각 최소 1개 포함해야 한다.
  - [ ] 비밀번호 확인 검증
    - 비밀번호 확인이 비어있는 경우
    - 비밀번호 확인이 비밀번호와 다른 경우
  - [ ] 닉네임 유효성 검증
    - 공백이 포함된 경우
    - 입력하지 않은 경우
    - 닉네임은 10자 이하여야 한다

### 회원정보 수정
- [ ] 이미지와 닉네임이 수정 가능하다

### 회원 삭제
- [ ] 회원 정보를 삭제한다

### 게시글 작성
- [ ] 제목, 내용, 이미지(선택)로 게시글을 작성한다
  - [ ] 제목 유효성 검증
    - 제목 길이는 26자 이하여야 한다
    - 입력이 비어있는 경우
  - [ ] 내용 유효성 검증
    - 내용이 비어있는 경우

### 게시글 상세 목록 조회
- [ ] 게시글 목록을 조회한다
- 제목, 좋아요, 댓글, 조회수, 작성일자, 작성자를 포함한다

### 게시글 상세 조회
- [ ] 제목, 작성자, 작성 날짜, 내용, 좋아요 수, 조회수, 댓글 수, 댓글을 조회한다

### 게시글 수정
- [ ] 제목, 내용, 이미지를 수정한다

### 게시글 삭제
- [ ] 게시글을 삭제한다

### 댓글 작성
- [ ] 게시글에 댓글을 작성한다

### 댓글 수정
- [ ] 게시글의 댓글을 수정한다

### 댓글 삭제
- [ ] 게시글의 댓글을 삭제한다
